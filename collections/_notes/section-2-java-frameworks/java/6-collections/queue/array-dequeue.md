* ArrayDeque creates a dynamic array and has no capacity restrictions. 
* Deque interface supports implementations with capacity restriction. 
* ***Default capacity is 16.***

```java
ArrayDeque()
ArrayDeque(int size)
ArrayDeque(Collection<? extends E> c)
```
