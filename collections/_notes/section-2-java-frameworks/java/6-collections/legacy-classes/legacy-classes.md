* Legacy classes are supported due to legacy code still in use.
* All the legacy classes are synchronized. 
* Use Collections class to get synchronized version.
* Legacy classes
	- Enumeration - Legacy interface. superseded by Iterator.
	- Dictionary
	- Hashtable
	- Properties
	- Stack 
	- Vector
