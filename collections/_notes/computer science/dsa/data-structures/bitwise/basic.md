**Bitwise AND**
```
0100_1011
0001_0101 &
----------
0000_0001
```

**Bitwise OR**
```
0100_1011
0001_0101 |
----------
0101_1111
```

**Bitwise XOR**
```
0100_1011
0001_0101 ^
----------
0101_1110
```

**Bitwise Complement**
```
0100_1011 ~
----------
1011_0100
```

**Bitwise Left Shift**
```
0100_1011 <<2
----------
0010_1100
```

**Bitwise Right Shift**
```
0100_1011 >>2
----------
??01_0010
```
? fill with bits, sign extended.
